# albert-spotlight-theme

Spotlight like theme for [Albert](https://github.com/albertlauncher/albert).

## Installing

 1. [Download zip](https://github.com/vinpac/albert-spolight-theme/archive/master.zip) and extract `Spotlight2.qss` to `/usr/share/albert/themes`.
 2. Restart albert if open.
 3. Click settings and change theem to Spotlight-2
 4. Be happy :)
